id

class
    chama no css com " . "
    lista de classes de um elemento, é um atributo global
    pode ser usada/chamada varias vezes no mesmo código
    classes permitem que CSS e JavaScript selecionem e acessem elementos específicos por meio dos seletores de classe ou funções como o método Document.getElementsByClassName ().

display-inline 
    tags <a> tem a propriedade display-inline
    permite que fiquem lado a lado

    a caixa ganha um comportamente de palavra, a tag a vira uma palavra só
    o que define o tamanho da caixa por padrão, é o conteudo
